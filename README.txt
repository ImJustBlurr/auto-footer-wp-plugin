== Description ==
Automatically places a customizable footer at the bottom of your post that will redirect users to your socials.

== Installation ==
1. Open config.php found in the root of this folder.
2. Edit the different components of the footer if you would like.
3. Upload the foler to the /wp-content/plugins/ directory
4. Activate the plugin in the Wordpress admin panel.
